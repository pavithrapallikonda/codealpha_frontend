# CodeAlpha - Image Gallery

## 📌 Project Description
A responsive Image Gallery created using HTML, CSS, and JavaScript with a lightbox feature.

## 🚀 Features
- Responsive Gallery
- Hover Effects
- Lightbox View
- Easy Navigation

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## ▶️ How to Run
1. Open the project folder.
2. Make sure the images are inside the `images` folder.
3. Open `index.html` in a browser.

## 📂 Project Structure
```
CodeAlpha_ImageGallery/
├── index.html
├── style.css
├── script.js
├── images/
└── README.md
```
