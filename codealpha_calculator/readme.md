# CodeAlpha - Calculator

## 📌 Project Description
This is a simple Calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations through an interactive user interface.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear Display
- Delete Last Character
- Keyboard Support
- Responsive Design

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## ▶️ How to Run
1. Download the project.
2. Open the project folder in VS Code.
3. Open `index.html` using Live Server or any web browser.
